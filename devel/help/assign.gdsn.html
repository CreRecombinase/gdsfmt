assign.gdsn {gdsfmt}R Documentation

Assign/append data to a GDS node

Description

Assign data to a GDS node, or append data to a GDS node

Usage

assign.gdsn(dest.obj, src.obj, append=TRUE)

Arguments

dest.obj

an object of class gdsn.class, a destination GDS node

src.obj

an object of class gdsn.class, a source GDS node

append

if TRUE, append data; otherwise, replace the old one

Value

None.

Author(s)

Xiuwen Zheng

References

http://github.com/zhengxwen/gdsfmt

See Also

read.gdsn, write.gdsn

Examples

# cteate a GDS file
f <- createfn.gds("test.gds")

n1 <- add.gdsn(f, "int", val=matrix(1:10000, nrow=100, ncol=100),
    compress="ZIP")

n2 <- add.gdsn(f, "int2", storage="int")

## Not run: 
assign.gdsn(n2, n1)

## End(Not run)

# close the gds file
closefn.gds(f)


# delete the temporary file
unlink("test.gds", force=TRUE)
